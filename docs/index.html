Statistical Methods in Psychological Research

All materials are totally free and open, including textbook, and lab manual.

This website can be copied

This website is written using R Markdown. The source code is here https://github.com/CrumpLab/psyc3400. For anyone interesting in adopting these materials, the idea is you can fork the repos for the website, the textbook, and the lab manual. Then edit as desired for your own purposes. This makes it easy to get many thing you might want to have all in one bundle, all ready to go.
