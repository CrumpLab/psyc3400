Copying/Adopting these materials

This website is written using R Markdown. The source code is here https://github.com/CrumpLab/psyc3400. For anyone interested in adopting these materials, the idea is you can fork the repos for the website, the textbook, and the lab manual. Then edit as desired for your own purposes.
